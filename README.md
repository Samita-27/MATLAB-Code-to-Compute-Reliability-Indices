📌 Project Title

Reliability Evaluation of Power Systems using MATLAB

📖 Description

This project presents a MATLAB-based implementation for computing power system reliability indices such as Loss of Load Expectation (LOLE) and Expected Energy Not Supplied (EENS). The analysis is performed using the Capacity Outage Probability Table (COPT) method.

The model can be applied to test systems such as the IEEE RTS 24-bus system and can be extended to include renewable energy sources like wind power.

🎯 Objectives
Compute key reliability indices (LOLE, EENS)
Model generator outages using COPT
Analyze the impact of generation adequacy
Provide a base framework for further research (e.g., wind integration, uncertainty modeling)
